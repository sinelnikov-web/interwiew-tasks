## Реализация функции каррирования

Необходимо написать функцию, которая бы принимала другую функцию и возвращала её каррированную версию.

```js
const sum = curry((a, b, c, z) => a + b + c + z);

console.log(sum(1)(2)(3)(4)); // 10;
console.log(sum(1)(2)(3, 4)); // 10;
console.log(sum(1)(2, 3, 4)); // 10;

```
